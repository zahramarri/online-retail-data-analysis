# 📊 Online Retail Data Analysis (End-to-End Project)

## 📌 Project Overview

This project is an end-to-end exploratory data analysis (EDA) and customer analytics case study based on an online retail transactional dataset.

The goal of this project is not only to analyze data, but to understand how raw transactional information can be transformed into meaningful business insights that support decision-making in areas such as customer retention, product strategy, and market focus.

---

## 🎯 Project Objective

The main objective of this project is to:

- Understand customer behavior through data
- Segment customers using RFM (Recency, Frequency, Monetary) analysis
- Identify high-value customers and product patterns
- Analyze revenue distribution across customers, products, and countries
- Translate analytical findings into actionable business insights

---

## 🧠 Key Business Questions

This project was structured around the following questions:

- Which customers generate the highest business value?
- Which products contribute most to revenue and demand?
- Which countries are the key markets for the business?
- Which customer segments should be prioritized?
- What strategic actions can improve business performance?
- What are the main insights from the overall analysis?

---

## 🛠️ Project Structure

The project consists of five main notebooks:

- **Notebook 01 — Data Understanding**
- **Notebook 02 — Data Cleaning**
- **Notebook 03 — Exploratory Data Analysis (EDA)**
- **Notebook 04 — RFM Customer Segmentation**
- **Notebook 05 — Business Insights & Strategy**

Each notebook builds upon the previous one to gradually move from raw data to business-level understanding.

---

## 📊 Key Techniques Used

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- RFM Customer Segmentation
- Grouped Aggregations (Revenue, Frequency, Monetary Value)
- Business Insight Generation

---

## 📌 Key Insights

- Customer value is highly concentrated in a small number of segments.
- A limited set of products drives a large portion of revenue and demand.
- Sales are unevenly distributed across countries, indicating market concentration.
- There is a clear difference between customer count, revenue contribution, and customer value.

---

## 💡 Strategic Takeaways

- High-value customer segments should be prioritized for retention strategies.
- Strategic products must be monitored and protected in inventory management.
- Marketing efforts should focus on both high-value segments and mid-value growth opportunities.
- Geographic dependency should be monitored to reduce market risk.

This project was built as a learning-focused data analysis case study.

A significant part of the implementation was developed with guidance and support during the learning process. The primary focus was not on writing every line of code independently, but on understanding each step of the analysis, learning how to structure a complete data project, and interpreting results from a business perspective.

Through this project, I learned:

- How to structure a full data analysis workflow from raw data to insights
- How to connect technical analysis with business questions
- How to perform customer segmentation using RFM methodology
- How to translate data outputs into business recommendations

At the same time, this project helped me realize areas I still need to improve, such as:

- Deeper understanding of business strategy and decision-making
- More advanced analytical modeling techniques
- Stronger ability to independently design end-to-end analytical systems
